# Security Policy

## Supported Yocto Versions

| Version                 | Supported          |
| ----------------------- | ------------------ |
| Yocto 4.x (Kirkstone)   | Yes                |
| Yocto 3.4 (Honister)    | EOL                |
| Yocto 3.3               | Untested           |
| Yocto < 3.3             | No                 |

## Reporting a Vulnerability

Please report vulnerabilities to the Eclipse Foundation Security Team at
security@eclipse.org
